# 1.2.5 Rich Picture

## O que é o Artefato Rich Picture?
Rich Pictures são artefatos que podem ser utilizados logo no começo do processo de elicitação. Trata-se de um artefato que não precisa ter orientação metodológica específica para usá-lo. Portanto, seja utilizando abordagens tradicionais, ágeis ou ainda mais emergentes, esse artefato é sempre bem vindo! Adicionalmente, tem-se que o cliente pode acompanhar esse artefato, dado que o mesmo não é demasiadamente técnico e/ou com notação complexa.
