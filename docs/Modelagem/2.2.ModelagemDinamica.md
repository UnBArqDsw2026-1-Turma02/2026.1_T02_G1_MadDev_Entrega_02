# 2.2. Módulo Notação UML – Modelagem Dinâmica

## O que é um Diagrama de Atividades?

Um Diagrama de Atividades é uma ferramenta UML que funciona como um fluxograma, focado em descrever os aspectos dinâmicos de um sistema, usando seus dados para ilustrar todas as possíbilidades de ações e consequências dessas ações como um diagrama de "passo a passo".
